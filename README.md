## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MariemBasily/movies_app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd movies_app
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
